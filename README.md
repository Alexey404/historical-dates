# Historical dates

## Description

A project on React TypeScript Webpack

### Installing dependencies

You can install dependencies using Yarn or npm.

#### With Yarn

```bash
yarn
```

#### With npm

```bash
npm install
```

### Launching a project in development mode

The project is launched using a dev script.

#### With Yarn

```bash
yarn dev
```

#### With npm

```bash
npm run dev

```
